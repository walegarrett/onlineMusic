# onlineMusic

## 介绍

本项目是本科android开发课程的期末大作业，项目的内容是在线音乐播放器。这个在线音乐播放器使用的是android studio完成开发，使用kotlin语言。本音乐播放器具有与市面上其他成熟的播放器一样的音乐播放功能，也包括在线音乐的搜索与播放。
